- [ ] Skill Name : (skill or spells name)
      Type\School : (Skill Types are Combat, Crafting, Passive, Activated, and Non-Combat. Spell Schools are Air, Earth, Fire, Water, Nature, Psionic, Arcane, Acid, Necrotic, Divine, and Shadow)
      Tier : (Default is 1, any skill with a Tier of 0 is selectable at character creation.)
      Description : What the skill does.
      Effects\Scaling : The mechanics of how the skill works and if it has higher tiers the changes for each tier.
      Prerequisites : Required Ability Scores, other skills, or previous tiers of the same skill.

- [ ] Skill Name : Animal Lore
      Type : Non-combat\Activated
      Tier : 1
      Description : This proficiency enables a character to observe the actions or habitat of an animal and interpret what is going on. Actions can show how dangerous the creature is, whether it is hungry, protecting its young, or defending a nearby den.
      Effects\Scaling : Allows the player to mimic the sounds of the animal within reason, to properly bait and conceal traps and snares, to approach the animal close enough to attempt to calm the animal.
      Prerequisites : Wisdom\15

- [ ] Skill Name : Animal Taming
      Type\School : Non-combat\Activated
      Tier : 1
      Description : Allows for the taming of calmed animals for training purposes.
      Effects\Scaling : increases Animal Training success chance by 50%
      Prerequisites : Animal Lore

- [ ] Skill Name : Animal Training
      Type : Non-combat\Activated
      Tier : 1
      Description : Players with this proficiency can train one type of Animal (declared when the proficiency is chosen) to obey simple commands and perform tricks.
      Effects\Scaling : Allows the player to train animals to perform certain tasks such as Mount, Guard animal, or Beast of Burden. The tasks that an animal can be trained for depends on the Animal type being trained. Untamed Animal has a 50% chance to be trained, a Tamed Animal has a 75% chance to be trained.
      Prerequisites : Animal Lore

- [ ] Skill Name : Summon Animal
      Type\School : Non-combat\Activated
      Tier : 1
      Description : Allows the player to summon an Animal that has been trained as a Mount, Guard animal, or a Beast of Burden. This animal is either one the player purchased from an NPC or PC.
      Effects\Scaling : Mount = Player is able to ride the animal if they have the appropriate riding skill.
                        Guard animal = The summoned animal will defend the player.
                        Beast of Burden = This animal will carry extra inventory for the player, up to the maximum that the animal is able to carry. Inventory is only accessible out of combat.
      Prerequisites : none

- [ ] Skill Name : Signature Animal\Monster
      Type\School : Non-combat\Activated
      Tier : 1
      Description : This is an Animal that the Player has tamed and trained. The player can equip this animal with Barding, and Trinkets to enhance the animals capabilities. This Animal is not summoned, but is always with the Player. This Animal earns XP equal to its Intelligence from the actions of the player.
      Effects\Scaling :  Mount = Player is able to ride the animal if they have the appropriate riding skill. Is allowed one Signature mount for each type of riding Animal\Monster skill known by the player.
                        Guard animal = The summoned animal will defend the player. The player may only have either a signature Guard Animal or Monster.
                        Beast of Burden = This animal will carry extra inventory for the player, up to the maximum that the animal is able to carry. Inventory is only accessible out of combat. The player may only have either a signature Guard Animal or Monster.

      Prerequisites : Animal Lore, Animal Taming, Animal Training, Monster Lore, Monster Taming, Monster Training
